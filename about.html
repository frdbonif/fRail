About fRail

fRail is designed to be a quick reference website with a basic, quick to load, mobile first interface. It is distributed through the BSD 2-Clause License meaning it is free for all to use, modify and distribute. The source code (HTML, CSS and JS) is available on our GitHub or can be accessed using your browser.


It uses a JavaScript lookup function to allow new pages to be built quickly by those with limited knowledge of web technologies. You can read more about the technical side on our Git repository.


The website is updated from our master branch daily. A testing version is also available at test-frail.fjla.uk
