contribute to fRail

fRail is available freely to all and everyone is welcome to contribute to the project on GitHub our repository is located at github.com/frdbonif/frail . It is easy to contribute to those with basic knowledge of HTML and JavaScript by using an existing 'page' as a template.


If you're not sure how to create a new page then you can add an issue on our GitHub page and a community member can build the new page for you.
